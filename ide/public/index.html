GPIO Register Map (0x2000_0000)

          

            
OffsetRegisterDescription

            
              
+0x00GPIO_INRead -> Switch inputs (16-bit)

              
+0x04GPIO_OUTWrite -> LED outputs (16-bit)

            
          


          
UART Register Map (0x3000_0000)

          

            
OffsetRegisterDescription

            
              
+0x00UART_TXWrite → Transmit byte

              
+0x04UART_RXRead → Received byte

              
+0x08UART_STATBit[0]=RX_VALID, Bit[1]=TX_FULL

            
          


          
Timer Register Map (0x4000_0000)

          

            
OffsetRegisterDescription

            
              
+0x00MTIME_LOLower 32 bits of the free-running timer

              
+0x04MTIME_HIUpper 32 bits of the free-running timer

              
+0x08MTIMECMP_LOLower 32 bits of compare register

              
+0x0CMTIMECMP_HIUpper 32 bits of compare register
